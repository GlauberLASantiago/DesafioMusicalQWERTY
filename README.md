# 🎼 Desafio Musical QWERTY

O **Desafio Musical QWERTY** é uma aplicação web interativa criada para treinar leitura musical de forma lúdica e dinâmica. O sistema apresenta notas no pentagrama e desafia o usuário a identificá-las rapidamente por meio do teclado do computador ou do piano virtual exibido na tela.

O jogo combina percepção visual, escuta, resposta rápida e memorização das notas, sendo útil para estudantes de música, professores e atividades de treinamento musical.

## ✨ Funcionalidades

- Jogo de identificação de notas musicais em tempo real
- Leitura em:
  - **clave de Sol**
  - **clave de Fá**
- Piano virtual interativo
- Resposta por:
  - teclado QWERTY
  - clique no piano virtual
- Reprodução sonora da nota exibida
- Barra de tempo para resposta
- Sistema de:
  - acertos
  - erros
  - nível
  - tempo de jogo
- Progressão por níveis
- Cenários com diferentes extensões de notas
- Ajuste do tempo máximo de resposta
- Trilha sonora de fundo opcional
- Seleção de timbre instrumental
- Ajuste de volume, reverb e sustentação
- Geração e cópia de relatório final
- Modo de alto contraste para acessibilidade

## 🎯 Objetivo do Projeto

O projeto foi desenvolvido para transformar o treino de leitura musical em uma experiência gamificada. A proposta é ajudar o usuário a:

- reconhecer notas no pentagrama;
- relacionar escrita musical e nome das notas;
- responder com mais rapidez e precisão;
- praticar leitura em diferentes regiões;
- fortalecer a percepção auditiva e visual.

## 🎮 Como funciona

Durante o jogo:

1. uma nota aparece no pentagrama;
2. o som correspondente é reproduzido;
3. o jogador deve identificar a nota antes que o tempo acabe;
4. a resposta pode ser dada pelo teclado QWERTY ou clicando no piano;
5. o sistema registra acertos, erros, nível e sequência de respostas.

## ⌨️ Teclas utilizadas

O jogo usa as teclas correspondentes às notas naturais:

- **C** → Dó
- **D** → Ré
- **E** → Mi
- **F** → Fá
- **G** → Sol
- **A** → Lá
- **B** → Si

## ▶️ Como usar

1. Abra o arquivo HTML em um navegador moderno.
2. Escolha o **cenário** desejado.
3. Defina o **tempo máximo de resposta**.
4. Informe seu nome, se quiser registrar o desempenho.
5. Clique em **INICIAR JOGO**.
6. Observe a nota no pentagrama e responda usando:
   - o teclado do computador
   - ou o piano virtual
7. Ao final, clique em **COPIAR RELATÓRIO** para salvar seu desempenho.

## 🎼 Cenários disponíveis

A aplicação oferece diferentes cenários para prática em regiões variadas do pentagrama, incluindo:

- clave de Sol em faixas médias
- clave de Sol em extensões mais amplas
- clave de Fá em regiões graves e médias

Isso permite adaptar o treino ao nível do estudante.

## 📈 Sistema de progressão

O jogo acompanha o desempenho do usuário com:

- **tempo total de jogo**
- **nível alcançado**
- **total de acertos**
- **total de erros**

A cada 10 acertos, o jogador sobe de nível.

## 🎧 Recursos sonoros

O projeto inclui:

- reprodução da nota mostrada no exercício;
- escolha de timbre General MIDI;
- trilha de fundo opcional;
- controle de volume da trilha;
- controle de volume das notas;
- ajuste de reverb;
- ajuste de sustentação.

## 📋 Relatório final

Ao final da prática, o usuário pode copiar um relatório contendo:

- nome do usuário
- cenário utilizado
- tempo de jogo
- tempo máximo de resposta
- nível alcançado
- acertos
- erros
- notas acertadas
- notas erradas
- uma citação inspiradora

## 🧩 Estrutura da Interface

A aplicação está dividida em áreas principais:

### 1. Cabeçalho
Contém o botão de iniciar/parar, seleção de cenário, tempo limite e acessibilidade.

### 2. Área do jogo
Mostra:
- o pentagrama
- a nota atual
- a barra de tempo
- estatísticas de desempenho
- painel de instruções

### 3. Configurações
Inclui:
- nome do usuário
- botão de copiar relatório
- trilha de fundo
- volume geral
- timbre
- reverb
- sustentação

### 4. Piano virtual
Permite resposta direta por clique.

## 🎨 Interface

A interface utiliza uma estética escura com destaques em azul, verde e vermelho, reforçando a sensação de jogo e resposta imediata. Entre os elementos visuais estão:

- pentagrama central com nota destacada;
- barra de tempo;
- feedback visual para acerto e erro;
- animação de subida de nível;
- piano virtual iluminado ao tocar.

## ♿ Acessibilidade

A aplicação inclui:

- **modo de alto contraste**
- boa separação visual entre áreas
- resposta multimodal por som, imagem e ação

A aplicação funciona melhor em:
